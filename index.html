Non-negative Kernel Graphs for Time-series Signals

Novel framework is proposed to represent sets of time-series signals using non-negative kernel (NNK) graph construction. The original NNK framework is extended to allow explicit delays as part of the graph construction. Propoesed framework is applied on EEG signals.

Non-negative Kernel Graph Based Interpretation of Deep Neural Networks

NNK algorithm is applied on the penultimate layer feature space of a neural network and polytopes are formed around each feature. Polytopes have geometric properties that are shown to be predictive in the neural network’s performance. Geometric properties
